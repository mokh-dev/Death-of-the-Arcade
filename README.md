# Death of the Arcade

This was a game was made for the FTC Jam in which it won first place. A big thanks to the FTC Jam Team for hosting the event.

Play the game: https://themkdev.itch.io/death-of-the-arcade


## Controls
AD - move left and right

Space - Jump

S - fast fall

Left Mouse Click - throw scythe in direction of mouse
